## Description

House Price Prediction project using Machine Learning. The project focuses on analyzing housing data, preprocessing it, and building models to predict house prices based on different features.

---

## README.md

# House Price Prediction

This project aims to predict house prices using Machine Learning techniques. It includes data preprocessing, feature engineering, and model building.

---

## Dataset

The dataset contains different features related to houses such as:

* Area
* Number of rooms
* Location
* Other relevant factors affecting house prices

---

## Project Steps

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Handling Missing Values
4. Feature Scaling
5. Model Training
6. Model Evaluation

---

## Models Used

* Linear Regression
*  Polynomial Regression  
* Decision Tree
* Random Forest
* Gradient Boosting
* ADABoost
* Ridge Regression
* Lasso Regression
* XGBoost

---

## Goal

To build a model that can accurately predict house prices based on input features.
